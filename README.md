EduLearn is a modern, human-centered online learning platform designed to bridge the gap between lecturers and students. It provides a seamless interface for managing course content, tracking student engagement, and fostering collaborative learning through interactive discussions.

🚀 Key Features

•
Dual-Role Dashboards: Tailored experiences for both Lecturers and Students.

•
Course Management: Effortlessly upload and organize materials including videos, documents, and quizzes.

•
Progress Tracking: Visual dashboards with animated progress bars and engagement metrics.

•
Interactive Discussions: A real-time discussion system for students to ask questions and instructors to provide guidance.

•
Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

•
Modern UI/UX: Clean, professional aesthetic with smooth animations and glassmorphism effects.

🛠️ Tech Stack

•
HTML5: Semantic structure for accessibility and SEO.

•
CSS3: Modern layout techniques (Flexbox, Grid) and custom animations.

•
JavaScript (ES6+): Dynamic interactions, tab switching, and notification systems.

•
Google Fonts: Utilizing Sora for headings and DM Sans for body text.
